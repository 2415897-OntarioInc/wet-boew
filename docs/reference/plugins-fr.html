Plugiciels

Needs translation A WET plugin provides various types of functionality useful to a website and provided by calling the function in the class attribute of an HTML tag.

  • (TODO) Augmentation des tableaux - Ajoutes des fonctionnalités aux tableaux telles que la recherche, le tri, le filtrage et la pagination.
  • (TODO) Barre de menu - Fournit une barre de menu avec des sous-menus facultatifs en forme de mégamenus.
  • (TODO) Calendrier d'événements - Permet de produire de façon dynamique une interface de calendrier pour parcourir une liste d'événements
  • Contenu archivé - Fournit un avis « archivé » pour votre contenu archivé. L’avis apparaît au haut de la fenêtre du navigateur lorsque l’utilisateur fait défiler vers le bas de la page.
  • (TODO) Expiration de la session - Aide les propriétaires d'actifs Web à configurer leurs actifs avec une fonction « Expiration de la session ».
  • (TODO) Formulaire de rétroaction - Permet aux utilisateurs de présenter leurs commentaires relativement à une page Web en particulier ou au site Web en général.
  • (TODO) Gadget de partage - Aide à partager le contenu Web dans divers résaux sociaux.
  • (TODO) Gadget des fils de syndication - Regroupe et affiche des données d'un ou plusieurs fils de syndication.
  • (TODO) Géocarte - Affiche une carte dynamique sur laquelle des informations de sources supplémentaires peuvent être superposées.
  • (TODO) Graphiques - Produit de façon dynamique des graphiques à l’aide des données figurant dans un tableau.
  • (TODO) Images réactives - Émule l’élément proposé picture (en anglais seulement) pour permettre que la source d'une image soit déterminée en fonction de Media Queries CSS.
  • (TODO) Interface à onglets - Convertit de façon dynamique plusieurs sections de contenu en une interface à onglets.
  • (TODO) Lecteur multimédia - Offre un lecteur multimédia qui permet d’intégrer dans des pages Web des fichiers vidéo et audio accessibles.
  • (TODO) Lightbox - Aide à construire galerie de photos à partir de photos sur une page Web.
  • (TODO) Mise en surbrillance de texte - Met automatiquement en surbrillance certains mots dans une page Web. La nature des mots ainsi mis en évidence peut être déterminée au moyen de la chaîne d'interrogation.
  • (TODO) Notes de bas de page - Met en place une méthode cohérente et facile d’accès pour répertorier les notes de bas de page dans l’ensemble des sites Web.
  • (TODO) Onglet coulissant - Affiche un onglet au bord de la zone de contenu avec la capacité de glisser pour afficher le contenu.
  • (TODO) Prettify - Mise en surbrillance de syntaxe des extraits de code source dans une page html en utilisant google-code-prettify.
  • (TODO) Rayage du zèbre - Permet l’ajout du rayage du zèbre aux listes et aux tableaux en colorant un élément de liste et une rangée de tableau sur deux.
  • (TODO) Sélecteur de langue - Permet d’établir des liens de façon automatique vers différentes versions linguistiques de la même page, dans les méthodes de conception ASP, Javascript, Perl et PHP.
  • (TODO) Validation de formulaires - Fournit la validation générique et le soutien de message d'erreur générique pour les formulaires Web.
